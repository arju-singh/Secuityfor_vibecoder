        How it works

        SentryScan is a non-intrusive tester. It loads pages the way a browser does and never attempts exploits. The render test uses a real headless Chromium to catch JavaScript errors. The code scanner analyzes files on the server and only contacts the public OSV.dev database to check dependency versions. Scans of localhost / private networks are blocked, and no results are stored.
